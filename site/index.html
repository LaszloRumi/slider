BoxSlider is a light-weight content slider for modern web applications

  • Multiple content and image slider effects
  • CSS 3D transitions
  • Easily customisable through options
  • Major framework/library support
  • Modern browser support
  • Accessibility baked in (need to actually add this!)
  • Light-weight, supports tree shaking
  • TypeScript support
  • Event API and Promises
  • Extensible, build your own transitions

Installation and usage

Install the BoxSlider package via NPM 

npm install --save @boxslider/slider

Alternatively using Yarn 

yarn add @boxslider/slider

Import the `BoxSlider` library with the desired effect and instantiate the slider.
